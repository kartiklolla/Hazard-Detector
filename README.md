# Hazard Detector 

Hazard Detector is an AI-Powered Safety Intelligence Tool designed to make Mining Safety details across India more accessible and actionable. From real-time pattern detection to automated safety audits along with latest news, 
the platform aims to keep users informed and compliant. The platform contains a variety of attractive features such as an AI Safety Officer, Location Intelligence and many more.

# Features

## Latest News & Updates : 
Stay informed with real-time news feeds highlighting the latest mining-related incidents across India.

## AI-Powered Safety Intelligence Tool :
Leverage a **Retrieval-Augmented Generation** (RAG)-based AI model trained on real **DGMS** (Directorate General of Mines Safety) data to answer queries, generate insights, and assist with safety audits.

## Location Intelligence :
Heat map provides insights into the severity of mining related issues based on regions for a larger look across India.
